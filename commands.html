Database Operations/ Commands

Creating Database

Create a new database with a unique name, which will store all the data for your application.

This operation is essential for setting up a new database management system.

After Creating Database

The database is created successfully, and you can now start creating tables and inserting data.

Creating Table

Create a new table with specified columns to store specific data.

This operation defines the structure of the data in your database.

Adding New Column

Add a new column to the existing table to store additional data.

This operation allows you to modify the structure of your table as needed.

Interface

Deleting New Column

Delete a column from the existing table to remove unnecessary data.

This operation helps maintain data consistency and reduces storage needs.

Deleting Table

Delete the entire table from the database to remove all associated data.

This operation is useful when you no longer need the data or want to start fresh.

Deleting Database

Delete the entire database to remove all associated tables and data.

This operation is useful when you want to completely remove a database management system.
