COMMANDS IN SQL

  1. CREATE:

    A SQL CREATE command is used to define a new database object such as a table, index, view, or stored procedure. It specifies the structure, columns, data types, constraints, and other properties for the object being created.

  2. DROP:

    A SQL DROP command is used to delete an existing database object such as a table, index, view, or stored procedure. This command removes the object and all its data from the database.

  3. ALTER:

    A SQL ALTER command is used to modify an existing database object such as a table, index, or view. This command allows you to add, delete, or modify columns, change data types, and apply other structural changes to the object.

  4. ADD:

    A SQL ADD command is used with the ALTER statement to add a new column or constraint to an existing table or database object.

  5. UPDATE:

    A SQL UPDATE command is used to modify existing data in a table. This command allows you to change the values of one or more columns for one or more rows based on a specified condition.

  6. SET:

    A SQL SET command is used to assign a value to a variable or to specify the value of a column in an UPDATE statement.

  7. INSERT:

    A SQL INSERT command is used to add new rows of data into a table. This command specifies the table, the columns, and the values to be inserted.

  8. SELECT:

    A SQL SELECT command is used to retrieve data from one or more tables. This command specifies the columns to be retrieved and can include conditions, sorting, and grouping to filter and organize the results.
