Create Gmail Account

Step Description
1 Go to Gmail website and click on "Create account"
2 Enter your first and last name, and choose a username (this will be your email address)
3 Enter a strong password and confirm it
4 Enter your recovery email and phone number (optional but recommended)
5 Enter your birthday and gender
6 Review the terms of service and privacy policy, then click "Next"
7 Verify your account through a text message or phone call (optional)

Create Github Account

Step Description
1 Go to Github website and click on "Sign up"
2 Enter your email address, choose a username, and enter a password
3 Enter your name and confirm your email address
4 Choose a plan (free or paid)
5 Review the terms of service and privacy policy, then click "Create account"
6 Verify your account through an email sent to your email address

Connect VSCode to Github

Step Description
1 Install the Github Extension for VSCode from the Extensions marketplace
2 Open VSCode and click on the Extensions icon in the left sidebar or press Ctrl + Shift + X (Windows/Linux) or CMD + Shift + X (Mac)
3 Search for "Github" and install the "Github" extension
4 Once installed, click on the Github icon in the left sidebar or pressCtrl + Shift + G (Windows/Linux) or CMD + Shift + G (Mac)
5 Sign in to your Github account using your credentials
6 You're now connected to Github from VSCode

Clone/Pull Repository from Github

Step Description
1 Open VSCode and click on the Explorer icon in the left sidebar or press Ctrl + Shift + E (Windows/Linux) or CMD + Shift + E (Mac)
2 Click on the "Clone Repository" button or press Ctrl + Shift + P (Windows/Linux) or CMD + Shift + P (Mac)
3 Enter the URL of the repository you want to clone (e.g., https://github.com/username/repository-name.git)
4 Choose a location to clone the repository to
5 Click "Clone" to clone the repository
6 Once cloned, you can pull the latest changes from the repository by clicking on the "Pull" button or pressing Ctrl + Shift + P (Windows/Linux) or CMD + Shift + P (Mac)

Push Repository/Folder to Github

Step Description
1 Open VSCode and navigate to the folder you want to push to Github
2 Initialize a new Git repository by running the command git init in the terminal
3 Add all files to the repository by running the command git add.
4 Commit the changes with a meaningful commit message by running the command git commit -m "Initial commit"
5 Create a new repository on Github and copy the repository URL
6 Add the remote repository URL to your local repository by running the command git remote add origin
7 Push the changes to Github by running the command git push -u origin master

Publish Repo as Live Site

Step Description
1 Create a new branch in your repository called gh-pages (this is the default branch for Github Pages)
2 Push the gh-pages branch to Github
3 Go to your repository settings on Github and click on "Pages" in the left sidebar
4 Select the gh-pagesbranch as the source branch
5 Choose a theme or upload your own HTML file
6 Click "Save" to save the changes
7 Your website is now live at <username>.github.io/<repository-name>
